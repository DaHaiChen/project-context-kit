# 事件契约

记录 MQ、领域事件、异步通知、定时任务触发等跨模块异步契约。

## 维护规则

- 新增事件时，说明生产者、消费者、触发时机、payload 和幂等要求。
- 修改事件 payload 时，说明兼容性影响。
- 消费者增加副作用时，说明失败重试和补偿策略。
